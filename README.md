## Installation

`go get github.com/gostudent/console`

## Usage

```go
package main

import (
	"github.com/gostudent/console"
)

func main() {
	console.Log("Hello World")
	console.Pizza("Tasty!")
	console.Beer("Cheers!")
	console.Poop("Oh F*ck!")
	console.Unicorn("So Magical!")
}
```
